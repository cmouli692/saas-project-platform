# SaaS Platform

A production-ready SaaS platform with authentication, project & task management.

## Features
- JWT authentication
- Projects & tasks management
- Pagination, search, sorting
- Soft delete & restore
- Rate limiting & security headers
- Request ID logging
- Swagger API documentation

## Tech Stack
### Backend
- Node.js
- Express
- PostgreSQL
- JWT
- Zod
- Docker

### Frontend
- React
- Redux Toolkit
- Tailwind CSS
- Axios

## Frontend Features
- Authentication (Redux Toolkit)
- Protected routes
- Axios interceptors
- Responsive layout (Tailwind)
- Sidebar + Navbar UI

- PostgreSQL
